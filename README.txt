Compiling and running:
----------------------
	Download the files and run the 'make' command on linux terminal in the containing folder. 
	The file 'shell.c' will be compiled and 'shell' is the executable. 
	The program can be run with './shell'. 

-------------------------------
1.Shell
-------------------------------
	Simple program designed to imitate the Linux shell; has an additional history feature.
